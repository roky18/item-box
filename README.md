## Project Name
Item-Box

## Description
A basic Next.js app with authentication and item management.

## Setup
npm install
npm run dev

## Routes
/ - Landing page
/login - Login
/items - Item list
/items/[id] - Item details
/add-item - Protected

## Features
- Mock Authentication
- Route Protection
- Express API Integration

## Technologies
Next.js 15
Express.js
Tailwind CSS
DaisyUI
